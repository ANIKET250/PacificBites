# 🌊 PacificBites Food Ordering App

## 📱 Project Overview
PacificBites is a modern Android food ordering application developed for the University of Fiji. The app allows users to order delicious Pacific cuisine including Chicken Palau, Lamb Palau, Pizza, and Chow Mein.

## ✨ Features Implemented

### Required Features (All Completed)
**Title** using TextView/Label
**Input fields** for:
    - Customer Name
    - Phone Number (10-15 digits validation)
    - Email Address (format validation)
**Dropdown list (Spinner)** with food items:
    - 🍗 Chicken Palau - $12.99
    - 🍖 Lamb Palau - $15.99
    - 🍕 Pizza - $10.99
    - 🍜 Chow Mein - $9.99
**Place Order button** with validation
**Email validation** with error messages
**Intent/Activity switching** to second screen
**Order confirmation screen** displaying:
    - Order Placed Successfully message
    - Customer Name
    - Phone Number
    - Email Address
    - Food Item Ordered
**Neat, user-friendly, properly aligned layout**
**Back to Home button** with form reset

### Additional Features
- 🎨 Modern gradient background design
- 📱 Responsive layout for all screen sizes
- ✨ Smooth animations
- 🛡️ Input validation for all fields
- 💬 Toast messages for user feedback
